## 8ï¸âƒ£ Behind the Scenes Tech

```javascript

4. Spawns 10 Micro worker processes (parallel!)
5. Each calls Cerebras LLM API
6. Results saved to rmri_contexts table
7. Meso worker processes Micro outputs
```
